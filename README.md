# Smart_Study_Planner
Overview:
The Smart Study Planner program is designed to help students manage their study time effectively. Instead of randomly deciding what to study, it creates a balanced schedule based on how difficult each subject feels to the user.
It ensures that tougher subjects get more attention while easier ones take less time. The program also saves the study plan so users can refer to it anytime.

Features:
1.Enter multiple subjects easily
2.Assign difficulty level (1 = Easy, 5 = Hard)
3.Automatically divides study time based on difficulty
4.Displays a clear and simple study schedule
5.Saves the schedule in a text file for later use
6.Helps in better time management and exam preparation

Technologies & Tools:
1.Built-in functions – Used for calculations and input/output
2.Text file storage – Saves study schedule (study_schedule.text)
3.Command-line interface – Simple and easy to use on any system

Installation & Running:
1.Make sure Python is installed on your system
2.Copy or download the program file
3.Open terminal or command prompt
4.Navigate to the folder where the file is saved
5.Run the program
6.Follow the on-screen instructions to create your study plan
